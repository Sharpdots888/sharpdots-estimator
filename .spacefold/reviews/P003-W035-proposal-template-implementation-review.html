How The Workflow Works

  • Load: selecting a starter and pressing Load Starter applies sanitized narrative fields and the matching publishing preset.
  • Source identity: the working proposal stores the original source template ID, source file, checksum, category, base preset, and source revision.
  • Rename: Rename Copy changes only the working copy display name. It does not change source ID, file, checksum, category, or preset identity.
  • Reset: Reset to Source reloads the selected source starter's reviewed narrative defaults.
  • Edit preview: Edit Copy makes title, subtitle, prepared for, overview, audience, deliverables, value narrative, next steps, and terms editable in the preview.
  • Locked sections: pricing, payment schedule, services, sourcing, print quote, and ecomm output remain calculated/read-only rendered sections.
